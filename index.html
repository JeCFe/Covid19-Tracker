Covid 19 Tracking

This project has been developed by retrieving and formatting data received from various API sources. Flagcdn, Map Quest Reverse Geocoding, Covid19 API and Rest Countries have all been used.

This covid tracker will use the geolocation given from the web browser to find the country being connected from, this will allow the users home country data to be first shown. By using ISO country codes all other country covid data is accessible.

This application is in early development, with many potential feature changes in the future.
